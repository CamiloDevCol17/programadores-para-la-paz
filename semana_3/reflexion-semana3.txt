Mantener un registro de cambios usando herramientas como Git es fundamental en el desarrollo de software porque nos brinda seguridad y control total sobre el código. Al programar, es extremadamente común cometer errores, borrar código accidentalmente o implementar nuevas funcionalidades que terminen rompiendo el proyecto principal. Al contar con un sistema de control de versiones, tenemos la tranquilidad de poder navegar por todo el historial del proyecto y revertir los cambios a un estado funcional sin perder nuestro progreso anterior. Además, en los entornos laborales reales, este registro es indispensable para facilitar el trabajo colaborativo, ya que permite que múltiples desarrolladores modifiquen distintos archivos del mismo proyecto de manera simultánea, evitando conflictos graves y asegurando que el trabajo de una persona no sobrescriba accidentalmente el esfuerzo de sus compañeros.
